# Radius Raid Gamelink #

This is an adaptation of the incredible Radius Raid videogame for the gamelink platform which allows you to play this game with a gamepad controller displayed on your smartphone. Gamelink is under active development and there is not a lot to show yet but I will add more info and a demo when it is ready!
